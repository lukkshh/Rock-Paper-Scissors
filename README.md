## Installation And Usage

```sh
git clone https://github.com/lukkshh/Rock-Paper-Scissors.git
```
```sh
cd  Rock-Paper-Scissors
```
```sh
pip install requirements.txt 
```
```sh
python game.py 
```
## Required Modules
- Tkinter
- Customtkinter
